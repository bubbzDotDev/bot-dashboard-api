# Announcement Bot Dashboard (API)

- [Link to Frontend repository](https://github.com/bubbzDotDev/bot-dashboard)
- [Link to Bot repository](https://github.com/bubbzDotDev/bot-dashboard-bot)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
